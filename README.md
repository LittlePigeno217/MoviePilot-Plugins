# 🎬 MoviePilot-Plugins

MoviePilot 第三方插件库。

> ⚠️ 当前仓库已整理为以 `plugins/` 为主的目录结构，旧的 `plugins.v2/` 目录已移除。

## 当前状态

当前仓库保留的实际插件目录为：

- `plugins/flztcheckin/`

插件元数据请查看：

- `package.json`

## 模板与开发

如果你要继续开发新插件，请查看：

- `templates/README.md`

其中包含：

- `templates/v2-vue-plugin/`
- `templates/v2-vuetify-plugin/`

## 说明

- `plugins/README.md` 为历史示例文档，主要提供外部消息转发的接入参考
- 若后续新增插件，建议同步更新 `package.json` 与本 README
